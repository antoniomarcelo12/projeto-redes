Aplicação:
	Foi desenvolvida uma calculadora remota, onde o cliente envia para o servidor os dados: tipo da operação(soma, subtração, divisão ou multiplicação), o primeiro numero e o segundo numero. O servidor realiza os calculos e envia o resultado para o cliente, que o mostra em tela.

	DNS:
		O servidor DNS desenvolvido possui uma lista de endereços disponíveis para assignar ao servidor que solicitar. Quando o cliente solicita ao servidor DNS o endereço de um dominio cadastrado, o servidor DNS retorna o endereço IP e porta relacionado ao domínio solicitado.
